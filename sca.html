Standards Compliant and Accessible website design

Standards Compliance

Every page of this website is written in valid XHTML 1.1 and CSS version 3 (W3C specifications) for separation of the website's content (the data) from the style (the layout). This is beneficial in a number of ways, including:

Robustness

The document must comply with a strict language specification, ensuring proper representation of data and separation of content from style, greatly reducing the chance of it "breaking".

Future compatibility

XHTML 1.1 is the newest mark-up specification and as such all modern internet devices are built first and foremost to interpret websites written in this way. Therefore compatibility with future browsers is maximised by using these standards.

Compatibility with mobile devices

A growing number of mobile devices can now access internet websites, but have limited browsing capabilities. Proper separation of content from style allows these devices to easily access the raw data without worrying about the layout.

Future Development

XHTML 1.1, being both valid HTML and valid XML, includes the capability to add new elements and modules, therefore maximising the ease with which the website can be upgraded to include new features.

You can check that each page of this website is valid XHTML 1.1 through the links on the site map (opens in parent window).

Accessibility

This whole website has been tested against the WebXACT verification tool for accessibility to people with disabilities and has passed all three priority levels and is therefore of WAI triple-A standard.

WebXACT is a well respected tool for checking websites for people with disabilities. With it web pages can be checked against the WAI standard for quality, accessibility, and privacy issues.

Access Keys

To improve the accessibility of this website access keys have been added for each page

To use the access keys:

  • In Internet Explorer hold down the 'alt' key and then press the access key. Then press 'enter'.
  • In Firefox hold down the 'shift' and 'alt' keys at the same time and press the access key.
